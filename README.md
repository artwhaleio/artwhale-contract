# artblock-marketplace-contracts

- Language: Solidity v0.8.10

- Project framework: hardhat + truffle / web3

- Nodejs: v14.17.0


## Installation & Usage

1. Install packages
```
npm i --save-dev
```

2. Build project
```
npm run build
```

### Testing

```
npm test
```

### Deploy

```
npm run scripts/<deployment script> --network <network name>
```

### Run linter

```
npm run lint
```
